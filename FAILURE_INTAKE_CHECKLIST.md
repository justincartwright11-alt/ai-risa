# AI-RISA Failure Intake Checklist

For any new failure, collect and record:

- [ ] Exact matchup ID
- [ ] Exact matchup file path
- [ ] Command run
- [ ] stderr/stdout output
- [ ] Emitted JSON path (if any)
- [ ] Failure layer (choose one):
    - input lookup
    - matchup module
    - engine pre-adapter
    - adapter
    - output

**No repair or patch work should begin until all checklist items are filled with real, file-backed evidence.**
